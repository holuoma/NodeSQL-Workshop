  • PATH /users
    • Defined in usersRoutes.js
    • GET /users/ Displays all users
    • GET /users/:id Gets a single user
    • POST /users/ Creates a single user (fields 'firstname' and 'lastname' are mandatory)
    • PUT /users/:id Updates a single user (fields 'firstname' and 'lastname' are mandatory)
    • DELETE /users/:id Deletes a single given user
  • PATH /
    • GET / Displays static html website explanation (this)
    • GET /static Static folder for html intro
    • GET /styles Static folder for css assets
  • PATH /demo/
    • Defined in seedRoutes.js
    • GET /seed/recreate Allows to seed the table with sample data
